# SauceDemo Playwright Automation Framework

## Overview
This project is an e2e test automation framework that was built using Playwright & Typescript. In this repo you will see that key user flows incldusing login, product selection, cart management, and checkout are covered.

## Features
- Page Object Mode (POM) design
- End-to-end checkout flow
- Cart validation and item removal
- Negative login test scripts
- Reusable and scalable test setup & structure

## Tech Stack
- Playwright
- Typescript
- Node.Js

## How to Run Tests

Tests can be run **either locally** from the terminal **or** **manually via GitHub Actions**.
